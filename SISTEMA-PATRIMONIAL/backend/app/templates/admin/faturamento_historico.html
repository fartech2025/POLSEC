Importar dados históricos de planilha

Para importar registros históricos de arquivos Excel (FATURAMENTO.xlsx, RES Estoque Polsec), execute o script na linha de comando do servidor: 

# Importação padrão (planilha com colunas UNIDADE, MÊS, ANO, TOTAL...)
python importar_faturamento.py FATURAMENTO.xlsx --tenant {{ tenant.slug }}

# Para planilha de mês único (sem coluna MÊS/ANO)
python importar_faturamento.py FATURAMENTO.xlsx --tenant {{ tenant.slug }} --mes 3 --ano 2026

# Para relatório de unidade única (ex: Penitenciária Franco da Rocha)
python importar_faturamento.py FATURAMENTO.xlsx --tenant {{ tenant.slug }} --mes 3 --ano 2026 \
       --unidade "PENITENCIARIA I FRANCO DA ROCHA"

# Simulação sem gravar
python importar_faturamento.py FATURAMENTO.xlsx --tenant {{ tenant.slug }} --dry-run
Execute a partir do diretório backend/. O script detecta automaticamente os cabeçalhos e vincula as unidades às filiais cadastradas.
