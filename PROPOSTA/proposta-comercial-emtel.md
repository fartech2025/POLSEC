## 1. Objetivo

Implantar um sistema web completo de controle patrimonial, substituindo o modelo atual baseado em planilhas por uma plataforma digital com banco de dados estruturado, automações e visão gerencial em tempo real.

O projeto visa garantir **controle, rastreabilidade, escalabilidade e segurança** das informações patrimoniais, além de preparar a empresa para evoluções tecnológicas futuras, como leitura por QR Code e auditoria digital.

---

## 2. Cenário Atual

O modelo atual baseado em planilha apresenta limitações que impactam diretamente a gestão:

- Alto risco de erro manual
- Dificuldade de auditoria e rastreabilidade
- Ausência de histórico estruturado
- Baixa escalabilidade
- Dependência de controle visual (cores)
- Ausência de indicadores gerenciais automáticos

---

## 3. Solução Proposta

A FARTECH propõe o desenvolvimento de um **Sistema Web de Controle Patrimonial**, com acesso online, multiusuário e integrado a banco de dados.

---

## 4. Escopo do Projeto

### 4.1. Sistema Web (Plataforma Online)
- Acesso via navegador (desktop e mobile)
- Login com controle de usuários e permissões
- Ambiente seguro e escalável
- Arquitetura moderna (SaaS-ready)

### 4.2. Banco de Dados Estruturado
- Cadastro completo de bens patrimoniais
- Armazenamento seguro e organizado
- Histórico de alterações (audit log)
- Rastreabilidade total por item

### 4.3. Módulo de Cadastro de Patrimônio
- Criação e edição de bens
- Categorização (tipo, setor, localização)
- Vínculo com responsável
- Status do bem:
  - Ativo
  - Em manutenção
  - Baixado
  - Extraviado

### 4.4. Gestão e Movimentação
- Transferência de bens entre setores
- Troca de responsável
- Registro de movimentações
- Histórico completo por item

### 4.5. Painel Gerencial (Dashboard)
- Total de ativos
- Distribuição por setor
- Bens por status
- Indicadores de controle
- Visão executiva em tempo real

### 4.6. Filtros e Busca Inteligente
- Busca por código, descrição ou responsável
- Filtros avançados
- Listagem dinâmica

### 4.7. Controle e Auditoria
- Registro de todas as ações realizadas
- Rastreabilidade completa
- Apoio para auditorias internas e externas

---

## 5. Diferenciais do Sistema

- Eliminação de controles manuais
- Padronização total das informações
- Acesso simultâneo por múltiplos usuários
- Aumento da confiabilidade dos dados
- Base preparada para integrações futuras
- Arquitetura pronta para expansão

---

## 6. Evoluções Futuras (Roadmap Tecnológico)

### Leitura de QR Code (Patrimônio Digital)
- Geração de QR Code por item
- Leitura via celular
- Identificação instantânea do patrimônio
- Acesso rápido às informações do bem
- Uso em auditorias e inventários físicos

### Inventário com Coletor Mobile
- Conferência patrimonial via smartphone
- Validação em tempo real
- Marcação de itens auditados
- Redução de tempo operacional

### Inteligência e Automação
- Alertas automáticos de inconsistência
- Sugestão de baixa ou manutenção
- Análise de uso e movimentação

### Integrações
- Integração com ERP ou sistemas internos
- API para expansão do ecossistema
- Conexão com outros módulos (financeiro, compras, etc.)
